\section{Das Humangenom-Sequenzierungsprojekt}\label{sec:das-humangenom-sequenzierungsprojekt}

\subsection*{Ablauf}\label{subsec:ablauf2}

\begin{itemize}
    \item Vermutete \gls{genom}größe:
    \[
        3000\,\mathrm{Mb} = 3 \times 10^{9}\,\mathrm{bp}
    \]

    \item Zwei konkurrierende Strategien:
    \begin{itemize}
        \item \textbf{Internationales Konsortium (öffentlich):}
        hierarchische Sequenzierung auf Grundlage physikalischer Karten
        (\gls{bac}-\gls{contig}s).

        \item \textbf{Celera Genomics (privat):}
        Whole-Genome-Shotgun-Sequenzierung.
    \end{itemize}

    \item Juni 2000:
    gemeinsame Vorstellung der \emph{Draft Sequence}.

    \item 2001:
    Publikation in
    \emph{Nature} (Internationales Konsortium)
    und
    \emph{Science} (Celera).

    \item 2003:
    offizieller Abschluss des Humangenomprojekts.

    \item 2006:
    letztes \gls{chromosom} vollständig sequenziert.
\end{itemize}

\subsection*{DNA-Herkunft}\label{subsec:dna-herkunft}

\begin{itemize}
    \item \textbf{Celera Genomics:}
    DNA von zwei europäischen Männern sowie je einer Frau aus Afrika,
    Asien und Mexiko (anonymisiert).

    \item \textbf{Internationales Konsortium:}
    DNA mehrerer Männer und Frauen (anonymisiert).

    \item Alle Individuen waren an zahlreichen Genorten heterozygot.

    \item Das Referenzgenom stellt daher ein
    \textbf{Mosaik verschiedener Individuen}
    dar und entspricht keiner real existierenden Einzelperson.
\end{itemize}

\subsection*{Warum war das Referenzgenom zunächst unvollständig?}\label{subsec:warum-war-das-referenzgenom-zunachst-unvollstandig?}

Der Draft des Jahres 2001 deckte im Wesentlichen den
\textbf{euchromatischen} Anteil des \gls{genom}s ab.

Nicht vollständig sequenziert waren:

\begin{itemize}
    \item \gls{heterochromatin}
    \item Telomere
    \item Subtelomere
    \item \gls{centromer} ($\alpha$-Satelliten)
    \item segmentale Duplikationen
    \item duplizierte \gls{genfamilie}n
    \item kurze Arme akrozentrischer \gls{chromosom}en
\end{itemize}

\paragraph{2020}

Telomere-to-telomere-Assembly des kompletten
\gls{x_chromosom}s (Miga et al.).

\paragraph{2022}

T2T-CHM13:

\begin{itemize}
    \item vollständige Sequenz aller \gls{chromosom}en außer Y
    \item ca. $3.055 \times 10^9$ Basenpaare
    \item Schließen der letzten etwa 8\,\% des \gls{genom}s
    \item etwa 200 Mb neue Sequenz
    \item 1956 neue Genvorhersagen,
    davon 99 proteinkodierende Gene
\end{itemize}

\subsection*{Syntenie}\label{subsec:syntenie}

\paragraph{Makrosyntenie}

\gls{gen}e eines einzelnen Mauschromosoms verteilen sich
auf mehrere menschliche \gls{chromosom}en.

Beispiel:

\[
    \mathrm{MMU1}
    \longleftrightarrow
    \mathrm{HSA1,\ HSA2,\ HSA5,\ HSA6,\ HSA8,\ HSA18}
\]

Die Orientierung kann erhalten oder invertiert sein.

\paragraph{Mikrosyntenie}

Über größere Bereiche bleibt die Reihenfolge der \gls{gen}e erhalten.

Beispiel:

\[
    \mathrm{HSA21}
    \longleftrightarrow
    \mathrm{MMU16,\ MMU17,\ MMU10}
\]

    Nachweis struktureller \gls{chromosomen_abberation}en,
    z.B. \gls{inversion}en oder \gls{translokation}en.
\end{itemize}

\section{Was sagt uns die Sequenz?}\label{sec:was-sagt-uns-die-sequenz?}

\subsection*{By the Numbers (DOE 2003)}\label{subsec:by-the-numbers-(doe-2003)}

\begin{itemize}
    \item Das humane \gls{genom} umfasst etwa
    \[
        3 \times 10^{9}
    \]
    Nukleotidbasen.

    \item Ein durchschnittliches \gls{gen} besitzt etwa
    \[
        3000\ \mathrm{bp}.
    \]

    \item Das größte bekannte \gls{gen} ist das
    \textbf{Dystrophin-Gen}
    mit einer Länge von ungefähr
    \[
        2.4 \times 10^{6}\ \mathrm{bp}.
    \]

    \item Die ursprüngliche Schätzung der Genzahl lag bei etwa
    30\,000 Genen
    (frühere Schätzungen gingen sogar von
    80\,000--140\,000 Genen aus).

    \item Heute wird die Zahl der proteinkodierenden \gls{gen}e auf
    etwa 20\,000--25\,000 geschätzt.

    \item Rund
    \[
        99.9\,\%
    \]
    aller Basen sind zwischen zwei Menschen identisch.

    \item Für mehr als die Hälfte aller Gene
    ist die biologische Funktion noch unbekannt.
\end{itemize}

\subsection*{Variation und Mutation}\label{subsec:variation-und-mutation}

\begin{itemize}
    \item Es wurden ungefähr
    drei Millionen SNP-Positionen
    (\emph{Single Nucleotide Polymorphisms})
    identifiziert.

    \item Verhältnis der Keimbahnmutationen:

    \[
        \text{männlich}
        :
        \text{weiblich}
        \approx
        2:1
    \]

    Zellteilungen während der Spermatogenese.

    \item Zusätzlich treten zahlreiche
    \textbf{Strukturvarianten} auf:

    \begin{itemize}
        \item Insertionen
        \item Deletionen
        \item Inversionen
    \end{itemize}

    Diese reichen von wenigen Kilobasen
    bis zu mehreren Megabasen.

    \item Kidd et al.
    (Nature, 2008):

    \begin{itemize}
        \item Untersuchung von acht menschlichen \gls{genom}en.
        \item Identifikation von 1695 Strukturvarianten.
        \item Nachweis von 525 neuen Insertionssequenzen,
        die im damaligen Referenzgenom fehlten.
    \end{itemize}

\end{itemize}

\section{Onkogene}\label{sec:onkogene}
\defin{onkogen}
\defin{proto_onkogen}

\subsection*{Der Weg zur Krebszelle}\label{subsec:der-weg-zur-krebszelle}

Tumorzellen entstehen schrittweise.

\begin{center}
    Primärzellen

    $\downarrow$

    begrenzte Zellteilungen

    $\downarrow$

    \textbf{Crisis}

    (Mehrzahl der Zellen stirbt)

    $\downarrow$

    \textbf{Immortalisierung}

    \begin{itemize}
        \item unbegrenzte Teilungsfähigkeit
        \item weiterhin adhärent
        \item serumabhängig
        \item kontaktinhibiert
    \end{itemize}

    $\downarrow$

    \textbf{Transformation}

    \begin{itemize}
        \item Verlust der Kontaktinhibition
        \item Wachstum ohne Serum
        \item Wachstum ohne Verankerung
        \item Fokusbildung
        \item Bildung solider Tumoren
    \end{itemize}

    $\downarrow$

    \textbf{Metastasierung}

    Mobilisierung der Tumorzellen und Besiedlung
    entfernter Gewebe.
\end{center}

\subsection{Retroviren als Vehikel}\label{subsec:retroviren-als-vehikel}


\subsubsection*{Entstehung viraler Onkogene}

Ein Retrovirus infiziert eine Wirtszelle
und übernimmt dabei ein zelluläres
Proto-Onkogen
(\textbf{c-onc}).

Dieses wird anschließend verändert
und liegt als

\[
    v\text{-}onc
\]

im Virusgenom vor.

Eigenschaften des viralen Onkogens:

\begin{itemize}

    \item intronlos

    \item entsteht über gespleißte \gls{mrna}

    \item kann zusätzliche Mutationen erwerben

    \item verursacht nach Integration
    Transformation der Zielzelle

\end{itemize}

Mechanismus:

\begin{enumerate}

    \item Deletion im Provirus

    \item Read-through-Transkription

    \item RNA-Splicing

    \item Verpackung ins Viruspartikel

    \item RNA-Rekombination
    mit einem Helfervirus

\end{enumerate}

\subsection*{Aktivierungsmechanismen von Proto-Onkogenen}\label{subsec:aktivierungsmechanismen-von-proto-onkogenen}

Proto-Onkogene können auf vier grundlegende Arten aktiviert werden.

\subsubsection*{1. Punktmutation}

Eine Punktmutation kann zu einem
konstitutiv aktiven Protein führen.

Beispiel:

\[
    RAS
    \longrightarrow
    \text{dauerhaft aktiv}
\]

%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%

\subsubsection*{2. Insertionsmutagenese}

Integration eines Retrovirus
führt zur Deregulation
eines Proto-Onkogens.

Möglichkeiten:

\begin{enumerate}

    \item
    Insertion innerhalb des \gls{gen}s

    $\rightarrow$

    Aktivierung codierender Exons
    durch Read-through-Transkription.

    \item
    Aktivierung eines
    \emph{fortuitous promoter}
    durch die virale LTR.

    \item
    Insertion hinter dem Gen

    $\rightarrow$

    Aktivierung des zelleigenen Promotors
    durch den viralen Enhancer.

\end{enumerate}

%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%

\subsubsection*{3. Chromosomale Translokation}

Ein Proto-Onkogen gelangt
unter die Kontrolle
eines stark aktiven Promotors.

Beispiel:

\[
    t(8;14)
\]

\[
    c\text{-}myc
    \longrightarrow
    IgH
\]

Burkitt-Lymphom
(B-Zellen)

\vspace{0.3cm}

In T-Zellen:

\[
    c\text{-}myc
    \longrightarrow
    TcR\alpha
\]

%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%

\subsubsection*{4. Genamplifikation}

Mehrere Kopien eines Proto-Onkogens
führen zu einer erhöhten Genexpression
und damit zur Tumorentstehung.

\subsection*{Mehrschrittmodell der Tumorentstehung}\label{subsec:mehrschrittmodell-der-tumorentstehung}

\begin{itemize}
    \item Die Krebsinzidenz steigt mit zunehmendem Alter stark an.
    \item Ursache ist die schrittweise Akkumulation mehrerer Mutationen.
\end{itemize}

Typischer Ablauf:

\[
    \text{Normales Epithel}
    \rightarrow
    \text{Erstmutation}
    \rightarrow
    \text{klonale Expansion}
    \rightarrow
    \text{weitere Mutationen}
    \rightarrow
    \text{Invasion}
    \rightarrow
    \text{Intravasation}
    \rightarrow
    \text{Metastasierung}
\]

Jede Mutation mit Wachstumsvorteil führt zu einer klonalen Selektion.

    \subsection*{Tumorsuppressorgen p53}\label{subsec:tumorsuppressorgen-p53}
    Wildtyp-p53 hemmt das Zellwachstum.

    Eigenschaften:

    \begin{itemize}
        \item Bindung an spezifische DNA-Sequenzen
        \item Aktivierung bzw.

        \item Zellzyklusarrest nach DNA-Schäden
        \item Einleitung der Apoptose
    \end{itemize}

    \begin{itemize}
        \item fehlende Aktivierung von Zielgenen
        \item erhöhte DNA-Amplifikation
        \item Ausbleiben des Zellzyklusarrests nach DNA-Schäden
        \item unkontrollierte Zellproliferation
    \end{itemize}

    \subsubsection*{Dominant-negativer Effekt}

    p53 bildet ein Tetramer.

    Bereits eine mutierte Untereinheit kann den gesamten Komplex funktionsunfähig machen.

    \subsubsection*{Domänen von p53}

    \begin{itemize}
        \item Transaktivierungsdomäne
        \item Signaldomäne
        \item DNA-Bindungsdomäne
        \item Oligomerisierungsdomäne
        \item C-terminale DNA-Bindungsregion
    \end{itemize}

    Mutationen konzentrieren sich überwiegend in der DNA-Bindungsdomäne.
\end{bsp}

\subsection*{Telomere und Telomerase}\label{subsec:telomere-und-telomerase}

\begin{itemize}

    \item Ohne \gls{telomerase} verkürzen sich die Telomere mit jeder Zellteilung.

    \item Dies führt zu

    \begin{itemize}
        \item Seneszenz,
        \item Crisis,
        \item Chromosomeninstabilität.
    \end{itemize}

    \item Reaktivierung der \gls{telomerase} regeneriert die Telomere
    und ermöglicht die Immortalisierung.

    \item Verkürzte Telomere führen zum Verlust von TRF2.

    \item Dadurch aktiviert p53 die DNA-Schadensantwort,
    was Zellzyklusarrest oder Apoptose auslöst.

\end{itemize}

\subsubsection*{Der Weg zur Krebszelle}

\begin{enumerate}

    \item Telomerverkürzung

    $\rightarrow$

    replikative Seneszenz

    $\rightarrow$

    Umgehung durch Ausschalten von Zellzyklus-Checkpoints.

    \item Early Crisis

    $\rightarrow$

    Aktivierung von p53

    $\rightarrow$

    Umgehung durch p53-Inaktivierung.

    \item Late Crisis

    $\rightarrow$

    genomische Instabilität

    $\rightarrow$

    Reaktivierung der \gls{telomerase}

    $\rightarrow$

    \textbf{Immortal Cell}

\end{enumerate}

\subsection*{Cancer Genomics}\label{subsec:cancer-genomics}

Der Fokus der Krebsforschung hat sich von einzelnen Genen hin zu genomweiten Analysen verschoben.

\begin{itemize}

    \item Whole-Genome-Sequenzierung von Tumor-Normal-Paaren

    \item Kombination mit

    \begin{itemize}
        \item Copy-Number-Analysen
        \item Expressionsprofilen
        \item Methylierungsprofilen
    \end{itemize}

    \item Identifikation bekannter Treibergene
    (z.\,B. \textit{TP53}, \textit{ARID1A}, \textit{CDH1})

    \item Entdeckung neuer Treibergene
    (z.\,B. \textit{MUC6}, \textit{CTNNA2}, \textit{GLI3}, \textit{RNF43})

    \item \textit{RHOA}-Mutationen treten bei etwa
    14,3\,\% diffuser Magenkarzinome auf.

    \item Besonders betroffen sind Signalwege der

    \begin{itemize}
        \item Adherens Junctions
        \item Focal Adhesions
    \end{itemize}

    \item Ziel ist eine genomgestützte personalisierte Tumortherapie.

\end{itemize}

\end{itemize}
