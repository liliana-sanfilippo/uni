    Sei $S_n$ die Anzahl der Erfolge in einem n-fach unabhГӨngig wiederholten Bernoulli-Experiment mit
    Erfolgswahrscheinlichkeit $p \in (0,1)$. \\
    Dann gilt fВЁur jede Wahl von Konstanten a, b mit $- \infty < a < b < \infty$
    \[
        \lim_{n \rightarrow \infty} \pe \big( a < \frac{S_n - np}{\sqrt{npq}} \leq b \big) = \frac{1}{\sqrt{2 \pi}} \intab e^{-x^{2} / 2} dx
    \]
    FВЁur groГҹe n folgt insbesondere, dass
    \[
        \pe(np - c < S_n \leq np +c ) \approx
        \txc{\int^{c / \sqrt{npq}}_{-c / \sqrt{npq}} \underbrace{ \frac{1}{\sqrt{2 \pi}} e^{-x^{2} / 2}}_{\vfktname} dx =}
        \frac{1}{\sqrt{2 \pi}} \int^{c / \sqrt{npq}}_{-c / \sqrt{npq}} e^{-x^{2} / 2} dx

    \txc{
        Das folgt direkt aus dem allgemeinen Satz, wenn man die Konstanten a und b passend wГӨhlt. \\
        Denn die WS fГјr  $np-c < S_n \leq np+c$ ist ГӨquivalent fГјr die von $-c < S_n - np \leq c$.
        Weil $\sqrt{npq} > 0$ (und das ist gegeben, weil $p \in (0,1)$, also
        $p$ ist echt grГ¶Гҹer als 0 und echt kleiern als 1;
        $n \leq 1$ ist die Anzahl der Versuche;
        $q = 1-p$ und wir wissen ja bereits, dass
        $p$ echt kleiner 1, also
        $q > 0$), kann man dadurch teilen und das hat dann die passende Form, um Moivre-Laplace anzuwenden:
        \[
            \frac{-c}{\sqrt{npq}} < \frac{S_n - np}{\sqrt{npq}} \leq \frac{c}{\sqrt{npq}}
        \]
    Und die Gleichung ergibt sich aus
    \[
        \vfkt \left(\frac{c}{\sqrt{npq}}\right) - \vfkt \left(-\frac{c}{\sqrt{npq}}\right) = \vfkt \left(\frac{c}{\sqrt{npq}}\right) - \left(1 - \vfkt \left(\frac{c}{\sqrt{npq}}\right)\right) = 2\vfkt \left(\frac{c}{\sqrt{npq}}\right) - 1
    \]
    }
   \begin{itemize}
       \item  Ist $c > 0$ nicht von n abhВЁangig, so geht die Ws. mit $n \rightarrow \infty$ gegen Null
       \txc{(c ist eine feste Konstante, hГӨngt nicht von nn
       n ab)}
       \item Ist $ 0 < c << \sqrt{n}$ , so geht die Ws. mit $n \rightarrow \infty$ immer noch gegen Null
       \txc{(c wГӨchst, aber langsamer als $\sqrt{n}$)}
       \item Relevanter Beitrag, d.h., Ws. grВЁoГҹer als null, nur fВЁur $c \approx \sqrt{n}$ (oder grВЁoГҹer)
       \txc{(c wГӨchst in derselben GrГ¶Гҹenordnung wie $\sqrt{n}$))}
   \end{itemize}
    \txc{
        Das erklГӨrt, wie sich die Wahrscheinlichkeit $\mathbb{P}(np-c < S_n \leq np+c) \approx 2\Phi\left(\frac{c}{\sqrt{npq}}\right) - 1$
        verhГӨlt, je nachdem wie schnell c mit n wГӨchst (oder eben nicht wГӨchst).
    }
\end{er}
