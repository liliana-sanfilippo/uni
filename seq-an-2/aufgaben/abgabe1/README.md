# SA2 Exercises

This is the repository for Sequence Analysis 2 programming exercises.
The actual exercises are described in the moodle.


## Python scripts
Python scripts are usually labeled `main_<xyz>.py`.

## Tests
Tests for libraries are usually labeled `test_<libname>.py`. You can simply run `python3 test_<libname>.py` and the tests for the library file `<libname>.py` will be executed.
These tests are not so much written as usual unit tests, but more as sanity checks for you to see if you're on the right track.


## TODOs
You can find the entry points for exercises via the TODOs in the corresponding python file.
