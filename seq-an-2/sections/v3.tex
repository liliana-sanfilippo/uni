A note on terminology: In Section~\ref{sec:the-q-gram-distance} we were speaking of q-grams when referring to
short (sub)strings of a certain length q.
This notation has originally been introduced in
the computer science literature and is there in use until today.
An alternative, equivalent
notion that one finds in most of the biological\footnote{The true origin is probably in biochemistry, where notions like monomer or polymer refer to molecular
chains of certain lengths.} and bioinformatics literature, is that of a
k-mer.
Here k refers to the (sub)string length.
To be more consistent with the literature,
in this chapter we will adopt that notation.

\section{Words with the same (k + 1)-mer Profile}\label{sec:words-with-the-same-k-1-mer-profile}
Motivated by the fact that the q-gram distance does not satisfy the identity property of
a metric (see Section~\ref{sec:the-q-gram-distance}) because there can be distinct sequences with the same q-gram
profile, we want to determine whether for a given sequence s and integer k there are other
sequences that are distinct but have the same (k + 1)-mer profile as s.
And, if there are,
how many? \\ \newline

For $k = 0$, the problem is trivial.
Indeed, if a sequence t that is distinct from s corresponds
to a permutation of the symbols of s, then s and t clearly share the same 1-mer profile.
For
$k \geq 1$, an elegant answer can be found with the help of the de Bruijn subgraph \dbg(s, k). \\ \newline
Clearly, by construction \dbg(s, k) forms an Eulerian path $p_s$ , represented by its sequence
of vertices
\[
    s_{k,1}, s_{k,2}, \dots, s_{k, |s| -k+1}.
\]
% TODO nochmal klären und erklären was dieses s-k+1 jetzt ist und woher das kommt
Consequently, \dbg(s, k) is connected and balanced
and might contain even more than one Eulerian path.
In fact, there is a one-to-one
correspondence between Eulerian paths in \dbg(s, k) and sequences sharing the same
(k + 1)-mer profile.
In other words, a sequence t that is distinct but has the same (k + 1)-
mer profile as s exists if and only if \dbg(s, k) has another Eulerian path $p_t$ orresponding
to the sequence of (k + 1)-mers of t.
Since $p_t$ uses the same edges as $p_s$ n a different
order, the path $p_t$ an only exist if \dbg(s, k) contains a cycle.
Note, however, that the
presence of a cycle is not a sufficient condition.
Indeed, it is possible that \dbg(s, k)
contains one or more cycles, but still admits only one Eulerian path.

\noindent We summarize the observations above in the following theorem.
\begin{theo}
    Given a sequence s and an integer $k \geq 1$, the number of distinct sequences that have the same (k + 1)-mer profile as s equals the number of distinct Eulerian paths in
    \dbg(s, k).
    If \dbg(s, k) is acyclic, it has a single Eulerian path, and no sequence t exists
    that is distinct but has the same (k + 1)-mer profile as s.
    \komm{Auf deutsch: \# worte mit gleichen (k+1)-mer Profil wie s = \# Euler-Pfade in \dbg(s,k)}
\end{theo}

distinguished in the de Bruijn graph.

A k-mer inherits this color, so that each k-mer
gets assigned a color set, indicating in which groups it appears and in which not.
